1-) Clonar o projeto Homogenise do branch feature_docker: https://github.com/evaldo/homogenise.git
Obs.: Para clonar utilize o seguinte comando -> git clone -b feature_docker https://github.com/evaldo/homogenise.git

2-) Caso necessário, instale o Python. Link para instalar o Python: https://www.python.org/downloads/

3-) Após instalação do Python, o pacote "pandas" tem que ser instalado para execução do sdd2rdf:
a-) Executar o "Prompt de comando"
b-) Instalar o pacote "pandas" usando o comando: pip install manager

4-) Importante configurar o caminho para execução do sdd2rdf pelo Python, por exemplo:
path C:\python39;c:\users\evald\appdata\local\programs\python\python37;C:\Users\evald\AppData\Local\Programs\Python\Python37\Lib\site-packages\pandas;C:\Users\evald\AppData\Local\Programs\Python\Python37\Lib\site-packages\numpy

5-) Dentro da pasta <unidade>\<pasta clonada>\homogenise, executar o sdd2rdf. Por exemplo:
python sdd2rdf.py config_quali.ini

Obs.: O grafo será gerado como extensão .TTL na pasta indicada na seção [Output] do arquivo .ini usado acima ->  <unidade>\<pasta clonada>\homogenise\sddquali\output\ttl\kg_quali.ttl

6-) Utilizar o programa Grapher para geração do grafo: http://www.ldf.fi/service/rdf-grapher

7-) Converter para Turtle: https://issemantic.net/rdf-converter
